<h1>Studypal-assistant - RAG-Powered AI Study Assistant</h1>



<h2>Description</h2>
RAG -powered AI study pal for student to learn any subject or chapter. This is build by integrating retrival augumented generation to build a question answering chatbot that can handle PDFs. Studypal helps students learn smarter by letting them pick a subject, choose a chapter, and instantly get AI powered explanation in simple language. It also provides video references and allows continuous chat like a personal tutor.
<br />

<h2>Features</h2>

- </b>User-Friendly interface</b> 
- </b>Subject Selection</b> 
- </b>AI-Powered Explanation</b> 
- </b>Vedio Reference</b>
- </b>Chat with history enabled</b> 



<h2>Tech Stack </h2>

- </b>Streamlit - for user Interface: To interact with chabot easily</b> 
- </b>Groq - LLM Provider: Grok is the LLM provider for fast and accurate answers.</b>
- </b>ChromaDB - Vector Store: To store and retrieve study materials.</b>
- </b>LangChain - Orchestration: To orchestrate all these components together</b>
- </b>AWS EC2 - Application Host: This is used to host the application in the cloud, so it can be shared with others/b>
